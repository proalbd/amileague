# আমিই আওয়ামী লীগ

বাংলাদেশ আওয়ামী লীগের "আমিই আওয়ামী লীগ" ছবি ফ্রেমের জন্য তৈরি করা ওয়েব অ্যাপ। এই প্রজেক্টটি একটি Progressive Web App (PWA), তাই এটি মোবাইল অ্যাপের মতো ইনস্টল ও ব্যবহার করা যায়।

## Live Demo

https://proalbd.github.io/amileague/

## ✨ Features
- সহজ ছবিতে ফ্রেম যুক্ত করা যায়
- দ্রুত লোডিং ওয়েব অ্যাপ
- মোবাইল ফ্রেন্ডলি ডিজাইন
- অফলাইন সাপোর্ট (PWA Cache)
- ইনস্টলযোগ্য (Add to Home Screen)
## PWA Support

এই প্রজেক্টে সম্পূর্ণ PWA সাপোর্ট রয়েছে:

- Service Worker ব্যবহার করা হয়েছে
- Static assets cache করা হয়
- Offline mode এ কাজ করে
- Manifest.json যুক্ত আছে
- Install prompt সাপোর্ট করে
